    Run dbench¶

    Install dbench on all client nodes:

    $ sudo yum install dbench

    From one of the client node:

    $ dbench --clients-per-process 10 --directory /tmp/daos_dfuse/ --loadfile /usr/share/dbench/client.txt --timelimit 10 10
