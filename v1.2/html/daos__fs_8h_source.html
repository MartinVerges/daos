47  * Consistency modes of the DFS container. A container created with balanced
48  * mode, can only be accessed with balanced mode with dfs_mount. A container
49  * created with relaxed mode, can be accessed with either mode in the future.

52  * for read / write access (O_RDONLY, O_RDRW).
